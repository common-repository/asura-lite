== Description ==

### Asura: the #1 WordPress plugin for selling Design sets built with Oxygen Builder.

Asura <a href="https://thelostasura.com/?utm_source=wprepo&utm_medium=link&utm_campaign=liteversion&utm_content=readme" rel="friend">[Pro]</a> provides a complete license key generation, activation, and checking system. 
It allows you to provide license keys that can then be used to access your design sets.

### Features details

= Licenses System =
Through simple, intuitive interfaces the Design set Licensing system allows site admins to easily view and manage customer license keys

= API System =
Asura's API System takes your licensing to a enterprise level. The API System is secure by design and extensible.

= License Generator <a href="https://thelostasura.com/?utm_source=wprepo&utm_medium=link&utm_campaign=liteversion&utm_content=readme" rel="friend">[Pro]</a> =
The generator allow us to bulk generate licenses and automatically with predefined configuration and design sets.

= Remotes <a href="https://thelostasura.com/?utm_source=wprepo&utm_medium=link&utm_campaign=liteversion&utm_content=readme" rel="friend">[Pro]</a> =
Asura remote feature is a highly efficient way to manage Asura on multiple websites from one place for which Asura is installed.

= WooCommerce and EDD integration <a href="https://thelostasura.com/?utm_source=wprepo&utm_medium=link&utm_campaign=liteversion&utm_content=readme" rel="friend">[Pro]</a> = 
Asura includes native integration for WooCommerce and Easy Digital Downloads by default, and many more integration soon. Selling design sets is easy!

== Frequently Asked Questions ==

= Does Asura Lite support integration with WooCommerce and Easy Digital Downloads? =

Asura Lite edition doesn't support integration with WooCommerce and Easy Digital Download.
The integration only available on [Pro](https://thelostasura.com) version.
